# Design Research Brief — AgentAuditor

## Reference Sites

### 1. Arkham Intelligence (intel.arkm.com)
- **Color palette:** Near-black (#0D0D0D) base, electric blue accents, white text, data-colored status indicators
- **Typography:** Clean sans-serif (Inter/Geist family), monospace for addresses/hashes
- **Layout:** Dense dashboard with sidebar nav, entity cards, bento-style data panels
- **Hero pattern:** No hero — direct dashboard entry, search bar prominent
- **Signature move:** Entity visualization with connected graph nodes, real-time data density
- **Animation:** Minimal, data-focused — number counters, subtle hover states

### 2. Nansen (nansen.ai)
- **Color palette:** Deep navy (#0A0E1A), cyan/teal accents (#00D1FF), gradient cards
- **Typography:** Modern geometric sans (close to Outfit/Inter), bold headings
- **Layout:** Card-based dashboard, portfolio view with sortable tables
- **Hero pattern:** Gradient mesh hero on marketing, minimal on dashboard
- **Signature move:** "Smart Money" labels with confidence indicators, wallet scoring
- **Animation:** Smooth transitions, skeleton loading, staggered card reveals

### 3. DeBank (debank.com)
- **Color palette:** True dark (#111111), green trust indicators, orange warnings
- **Typography:** System sans, compact, monospace for values
- **Layout:** Compact data-dense panels, horizontal chain tabs
- **Hero pattern:** Profile-centric — address input leading to wallet dashboard
- **Signature move:** Trust score with visual breakdown bars, chain switcher pills
- **Animation:** Number count-up, progress ring animations

### 4. Dune Analytics (dune.com)
- **Color palette:** Dark purple-gray (#1A1A2E), orange accent (#FF6B2C), data viz colors
- **Typography:** Clean sans, code blocks in monospace, clear hierarchy
- **Layout:** Full-width data viz, SQL editor + chart combo, dashboard grid
- **Hero pattern:** Query-first — search/input as the entry point
- **Signature move:** Beautiful data visualization with custom color scales
- **Animation:** Chart drawing animations, smooth tab transitions

### 5. Chainalysis Reactor
- **Color palette:** Professional dark (#1A1D21), blue institutional accents, status colors
- **Typography:** Professional sans-serif, dense but readable data tables
- **Layout:** Investigation-style — graph visualization + detail panels
- **Hero pattern:** Tool-first, minimal chrome, maximum data
- **Signature move:** Transaction graph with directional flow arrows
- **Animation:** Graph node animations, data loading skeletons

## Design Intelligence Summary

### Common Patterns (to learn from)
- Dark backgrounds between #0A0A0A and #1A1A2E
- Green = safe/positive, Yellow = caution, Red = danger (universal)
- Monospace for addresses, hashes, values
- Score gauges (ring/radial) for trust/confidence
- Dense data tables with explorer links
- Search/input as primary entry point
- Chain selector pills/tabs

### Differentiators (to aim for)
- Arkham's entity visualization depth
- Nansen's polish and gradient work
- DeBank's compact efficiency
- Dune's data visualization beauty
- Chainalysis's professional authority
